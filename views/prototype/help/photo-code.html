How to get a photo code

Get a photo taken in a booth or shop, and choose to get a code with your photo.

How to use a photo code

It's simple to use the code to retrieve your digital photo. You need to:

  1. Start your online passport application
  2. In the photo section, select the option enter the photo code
  3. After entering the code, you'll see your photo on the screen
  4. You'll be asked to check your photo, and given the option to submit it
  5. Your photo will then be added directly to your application
